工具链安装OpenOCD

  • ARM-GCC
  • gdbgui[Linux]
  • make[Win]
  • MSYS2[Win]
  • OpenOCD
  • TDM-GCC[Win]
    • Jump to Section
    sudo apt install openocd

    备选方案是使用MSYS2安装。需要注意的是，在MSYS2中安装的工具链，要在MSYS2的MINGW终端中使用。

    下载

    • http://openocd.org/getting-openocd/

    安装

    直接安装，并将\bin目录添加进入环境变量PATH即可。
